# AI智能打印助手 - 微信小程序 PRD

## 1. 产品概述

### 1.1 产品定位
适配智能AI打印机的微信小程序，面向学生和家长群体，提供语音交互生成学习内容、一键打印的服务。

### 1.2 目标用户
- **主要用户**：小学生家长（25-45岁）
- **次要用户**：中小学生（8-15岁）
- **使用场景**：家庭作业打印、课外练习生成、学习资料整理

### 1.3 核心价值
- 语音说话即可生成习题，零门槛操作
- AI智能生成口算题、生字表、单词听写等学习内容
- 一键发送到智能打印机，无需电脑

---

## 2. 功能架构

### 2.1 功能模块总览

```
AI智能打印助手
├── 首页（语音交互 + 快捷入口）
├── 知识库（内容归档管理）
├── 打印记录（打印任务追踪）
├── 我的（打印机管理 + 设置）
└── 内容预览页（编辑 + 发送打印）
```

### 2.2 页面清单

| 页面 | 路径 | 说明 |
|------|------|------|
| 首页 | `/` | 语音按钮、问候语、快捷功能、Tips |
| 知识库 | `/knowledge` | 内容归档、统计、列表 |
| 打印记录 | `/print` | 打印任务状态追踪 |

| 手动输入 | `/manual` | 文本/图片/文档上传 |
| 内容预览 | `/preview` | 内容编辑、发送打印 |

- **按下态**：scale(0.98) 缩小

### 5.4 列表交互
- **卡片点击**：scale(0.99) 微缩 + 边框色变化
- **滑动操作**：暂不支持

## 6. 数据模型

```typescript
{
  id: string;
  type: 'math_arithmetic' | 'math_word' | 'chinese_chars' | 
        'chinese_practice' | 'english_words' | 'custom_text' | 
        'image' | 'document';
  title: string;
  content: string;
  metadata: {
    grade?: string;        // 年级
    subject?: string;      // 科目
    count?: number;        // 数量
    range?: string;        // 范围
    operation?: string;    // 运算类型
    size?: number;         // 文件大小
    previewUrl?: string;   // 图片预览URL
    mimeType?: string;     // 文件类型
    fileName?: string;     // 文件名
  };
  createdAt: number;       // 创建时间戳
  status: 'generated' | 'editing' | 'sent' | 'printing' | 'printed';
}
```

### 6.2 打印任务 (PrintJob)
```typescript
{
  id: string;
  contentId: string;
  title: string;
  type: GeneratedContent['type'];
  status: 'queued' | 'sending' | 'printing' | 'completed' | 'failed';
  progress: number;        // 0-100
  createdAt: number;
  completedAt?: number;
}
```

### 6.3 打印机设备 (PrinterDevice)
```typescript
{
  id: string;
  name: string;
  model: string;
  firmware: string;
  connectionType: 'Wi-Fi 直连' | '蓝牙' | 'Wi-Fi';
  ipAddress?: string;
  inkLevel: number;        // 0-100
  paperRemaining: number;  // 剩余张数
  paperSize: string;       // A4, A5, B5...
}
```

### 6.4 打印机状态 (PrinterState)
```typescript
{
  status: 'connected' | 'disconnected' | 'connecting' | 'scanning';
  device: PrinterDevice | null;
  discoveredDevices: PrinterDevice[];
  wifiConfig: { ssid: string; password: string } | null;
}
```

## 7. 语音指令解析规则

### 7.1 内容类型识别
| 关键词 | 类型 |
|--------|------|
| 英语、单词、听写 | english_words |
| 生字、字表 | chinese_chars |
| 练字、字帖 | chinese_practice |
| 应用题、文字题 | math_word |
| 其他（默认） | math_arithmetic |

### 7.2 参数提取
| 参数 | 识别规则 | 默认值 |
|------|---------|--------|
| 年级 | 初[一二三] / [一二三四五六七八九123456789]年级 | 当前设置年级 |
| 数量 | (\d+)\s*(道\|题\|个\|词\|组) | 50（口算）/ 20（其他） |
| 范围 | \d+\s*以内 | 10以内 |
| 运算 | 乘法/除法/加减 | 加减法 |

### 7.3 示例指令
| 用户说 | 解析结果 |
|--------|---------|

| `/api/voice-parse` | POST | 语音指令解析 | `{ text: string }` |
| `/api/asr/xfyun-url` | GET | 获取讯飞ASR鉴权URL | - |

## 9. 非功能需求

### 9.1 性能要求
- 首屏加载时间 < 2秒

- AI内容生成时间 < 5秒
- 页面切换流畅，无卡顿

- 微信小程序基础库 2.25.0+
- iOS 12+ / Android 7+

### 9.3 安全要求
- 用户数据本地存储，不上传云端
- 打印机连接信息加密存储
- API接口鉴权
