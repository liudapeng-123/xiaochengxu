# DESIGN.md

## 项目与用户画像
- 产品：智能AI打印机配套移动端应用（微信小程序风格）
- 用户：学生、家长，追求极简操作体验
- 场景：语音指令生成学习内容，一键打印

## 品牌与视觉方向
- 风格：极简、清新、亲和，适合家庭/教育场景
- 主色调：柔和蓝绿色系，搭配暖色点缀
- 整体感觉：微信小程序原生风格，圆角卡片，大按钮，留白充足

## Design Tokens

### 色彩
- 主色：#59B0FE（柔和蓝，用于语音按钮、主操作按钮）
- 主色渐变：#59B0FE → #6AC4FE（语音按钮渐变、进度条）
- 辅助色：#98E4AD（薄荷绿，用于成功状态、已完成）
- 辅助色：#F7A38D（暖橙粉，用于进行中、警告）
- 辅助色：#C197FA（淡紫，用于心理/情感类）
- 辅助色：#84C5F9（淡蓝，用于通用/设置类）
- 背景渐变：linear-gradient(135deg, #E2E2F9 0%, #F7FBFF 35%, #BCE2FD 100%)（淡紫蓝 → 淡天蓝）
- 卡片色：#FFFFFF（纯白，带柔和弥散阴影）
- 卡片阴影：0 10px 35px rgba(144,176,208,0.10)
- 文字主色：#2B333F（深灰蓝）
- 文字次色：#6B7A90（中灰蓝）
- 边框色：#E8ECF0
- 成功色：#98E4AD（薄荷绿）
- 警告色：#F7A38D（暖橙粉）
- 错误色：#EF4444

### 功能卡片配色（马卡龙色系）
- 淡蓝底：#E0F0FF + 文字 #4A8FD4（通用/文本）
- 淡粉底：#FDE8E4 + 文字 #D47A5A（提醒/活力）
- 淡紫底：#F0E8FF + 文字 #8A6FD4（心理/情感）
- 淡绿底：#D1F5E1 + 文字 #3D7A5A（记录/健康）

### 字体
- 字体族：-apple-system, BlinkMacSystemFont, 'PingFang SC', 'Helvetica Neue', sans-serif
- 标题字号：18px-20px
- 正文字号：14px-16px
- 辅助文字：12px

### 圆角
- 小圆角：8px（按钮、输入框）
- 中圆角：12px（卡片）
- 大圆角：20px（语音按钮、底部弹窗）
- 全圆角：50%（圆形按钮）

### 间距
- 页面边距：16px
- 卡片内边距：16px-20px
- 元素间距：12px-16px

### 阴影
- 卡片阴影：0 10px 35px rgba(144,176,208,0.10)（柔和弥散，带蓝色调的灰）
- 按钮阴影：0 4px 12px rgba(79,142,247,0.3)

## 布局与响应式
- 移动端优先，最大宽度 480px 居中
- 底部 Tab 导航（首页/打印记录/我的）
- 首页：顶部状态栏 + 中央语音按钮 + 快捷功能网格
- 内容卡片式布局，列表式展示

## 交互与状态
- 语音按钮：按下态放大 + 波纹动画 + 颜色变化
- 页面切换：左右滑动过渡
- 加载状态：骨架屏 + 脉冲动画
- 打印状态：进度条 + 状态图标变化
